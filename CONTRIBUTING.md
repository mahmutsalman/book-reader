# Contributing to Smart Book Reader

First off, thank you for considering contributing to Smart Book Reader!

## How Can I Contribute?

### Reporting Bugs
* Check the [Issues](https://github.com/mahmutsalman/book-reader/issues) tab to see if the bug has already been reported.
* If not, open a new issue. Include a clear title, a description of the problem, and steps to reproduce it.

### Suggesting Enhancements
* Open an issue with the "enhancement" tag.
* Describe the feature you'd like to see and why it would be useful.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure your code follows the existing style of the project.
4.  Issue a pull request to the `main` branch.

## Development Setup
1.  Clone the repository.
2.  Install dependencies: `npm install` (or the relevant command for your environment).
3.  Run the development build: `npm run dev`.

## Code of Conduct
Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
